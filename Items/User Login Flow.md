## User Story

**As a** returning user
**I want** to log in with my email/password or OAuth
**So that** I can access my dashboard and data

## Acceptance Criteria

- [ ] Login form validates email format
- [ ] Error messages are clear and helpful
- [ ] OAuth buttons redirect to correct providers
- [ ] Successful login redirects to dashboard
- [ ] Failed login shows appropriate error

## Notes

Keep the UI minimal. Single column, centered, with clear CTA.
